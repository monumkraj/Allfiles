History of Html

The first version of HTML was written by Tim Berners-Lee in 1993.
Since then there have been many different versions of HTML.

The most widely used version
throughout the 2000's was HTML 4.01, which became an official standard in December 1999.
Another version, XHTML, was a rewrite of HTML as an XML language

Tim Berners-Lee

Sir Timothy John Berners-Lee OM KBE FRS FREng FRSA DFBCS RDI (born 8 June 1955), [1] also known as TimBL, is an English computer scientist best known as the inventor of the World Wide Web, the HTML markup language, the URL system, and HTTP.

HTTP

The Hypertext Transfer Protocol (HTTP) is the foundation of the World Wide Web, and is used to load webpages using hypertext links. HTTP is an application layer protocol designed to transfer information between networked devices and runs on top of other layers of the network protocol stack.
